# Tripleten web_project_around

o projeto é uma pagina para representar uma rede social, foram implementadas funções como formulário, programação com java script (abre formulário quando clicamos no botão editar, preenche campos de nome e função, imprime na página essas novas informações quando clicamos em salvar).

Manipulação do DOM com querySelector() e addEventListener()
Eventos de formulário (submit, click)
Modais/Pop-ups com abertura e fechamento dinâmicos
Atualização de conteúdo em tempo real
